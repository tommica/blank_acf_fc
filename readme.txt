== Description ==

This is a theme based on the "blankslate" theme (https://github.com/tidythemes/blankslate) which is modified to support the "Flexible Content" feature from "Advanced Custom Fields".

Just search-replace all "blank_acf_fc" strings in this theme with your themes name.

To register a new content block:
    - Add a new element to the "flexible content" custom field
    - Add a template for it in the "flexible-content" folder in the theme
    - Add a new line to the $blocks variable in "acf-content.php" file
